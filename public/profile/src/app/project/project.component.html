EnterpriseAPITemplate

This is the enterprise level of building applications which follows design patterns to separate Data Layer from Presentation Layer(Controller) and Model

  • Controller Layer
  • Data Access Layer
  • Model Layer
  • Service Layer
  • Utility Layer
  • DaoFactories class to Get a provider specific (i.e. database specific) factory
  • DataAccessObjects This class shields the client from the details of database specific data- access objects. It returns the appropriate data-access objects according to the configuration.
  • ValidatorRule abstract class is to maintain property name to which rule applies and validation error message.
